# AI駆動開発における FXシステム向けレビュー・監査チェックリスト 表形式転記版

## 1. 目的

本資料は、Excel、Google Sheets、社内表計算フォーマットへ転記しやすいよう、チェック項目を行単位で並べた版である。  
レビュー会、監査記録、リリース判定表へそのまま移しやすい構造にしている。

---

## 2. 列定義

推奨列:

| 列名 | 内容 |
|---|---|
| ID | チェック項目ID |
| 区分 | 業務整合性、境界、補償、実装品質など |
| チェック項目 | 確認したいこと |
| 確認観点 | 何を見て判断するか |
| 判定 | `OK / 要確認 / NG / N/A` |
| コメント | 懸念、補足、条件付き事項 |
| 根拠資料 | 設計書、PR、レポート、ログなど |
| 対応者 | 修正または確認担当 |
| 期限 | 対応期限 |

## 4. フィルタ例

- `NG` のみ抽出: 即対応が必要な項目
- `要確認` のみ抽出: 条件付きGoの論点
- `区分=コンプライアンス` で抽出: 監査向け確認
- `区分=テスト` で抽出: 検証不足の洗い出し
